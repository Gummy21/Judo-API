# Judo-API
The API of the Judo app.

## General info

The Back-end of the Judo movelist app, built using NodeJS, Express, Sequelize and a MySql database.

## Technologies
Created with:
* Express: 4.17.1
* Sequelize: 6.6.0
* NodeJS: v14

## Setup 
To run this project, install it locally using npm:
```
$ cd ../Judo-API
$ npm install
$ npm start
